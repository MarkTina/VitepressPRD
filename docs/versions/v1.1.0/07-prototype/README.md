## 使用说明

- 上方表格中的链接会在新窗口打开高保真 H5 原型，基于 Vue 3 + Tailwind CSS Play CDN，单文件部署在 `docs/public/prototype/` 下。
- 桌面端（≥1024px）右下角悬浮按钮可切换「全局模式」，一屏查看所有页面缩略图，滑块可调节缩放。
- 移动端自动切换抽屉式侧栏和卡片堆叠列表视图，无横向溢出。
- 所有数据均为 Mock，可在原型内自由新增 / 编辑 / 绑定 / 调整配额而不会影响真实系统。
- 新增静态原型素材时，将文件放入 `docs/public/prototype/<编号>/` 目录，并在素材索引表中登记。
