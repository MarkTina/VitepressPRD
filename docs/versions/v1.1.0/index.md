## 变更日志

### 新增

- [短信微服务] 将快递业务中的短信发送模块拆离为独立微服务，支持多租户、模板管理、异步发送与状态回调
- [支付配置中心] 新建网点级支付配置管理，支持创建多个收款配置，每个网点可指定独立的收款模式
- [微信分账] 服务商模式下微信支付分账能力，支持按比例/固定金额分账给多个接收方
- [服务商直收] 服务商模式下微信支付直接收款，支持子商户进件与管理
- [支付宝直收] 支付宝直接收款模式，独立于微信支付体系

### 优化

- [支付解耦] 支付层从洗衣业务中剥离，形成独立的支付网关层
