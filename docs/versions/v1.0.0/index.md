- [组件] 自定义 Vue 组件（PrdMeta、StatusBadge）
- [脚本] new-version、new-feature CLI 工具
- [AI] 4 阶段 AI Prompt 模板（需求讨论 → PRD 生成 → 审校 → 用户文档）
