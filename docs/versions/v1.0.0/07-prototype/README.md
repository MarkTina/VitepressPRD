## 使用说明

1. 将设计稿截图文件放入此目录
2. 在素材索引表中登记编号、名称、关联功能
3. 在对应功能模块的文档中引用素材编号
4. 建议使用统一命名规范：`{编号}-{功能名}-{描述}.png`
